# 从 0 开始重建方案（Spring Boot + 高级前端）

## 当前已落地

- 后端：Spring Boot 3（内存数据版）
  - 登录接口与鉴权拦截
  - 老师题库管理（新增/编辑/删除/筛选）
  - 手动组卷、随机组卷
  - 平行组创建、随机发放
  - 发布考试、查看提交
  - 学生查看考试详情并提交
  - 客观题自动判分
  - 统计 API
- 前端：
  - 教师端高级工作台（登录、题库管理、组卷、平行发放、发布考试、提交记录查看、统计）
  - 学生端作答页面（考试列表、载入题目、在线提交）

## 技术路线

### 后端

- Spring Boot 3.3 + Java 17
- Bean Validation 参数校验
- 全局异常处理 + CORS + 登录拦截器
- 下一步建议：MySQL + MyBatis-Plus + Redis + JWT

### 前端

- 当前：原生 HTML/CSS/JS（快速可运行）
- 下一步建议：Vue3 + TS + Pinia + Element Plus + ECharts
